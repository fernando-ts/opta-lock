What is it and how does it work?

Optalock MDM is a solution for managing corporate-owned Android devices.With the help of an Android Launcher installed on an empty device with extended privileges.

  • Single Tasks Mode (Kiosk Mode)
  • GPS tracking
  • Remote access, lock and unlock
  • Decide and limit the applications to install on your corporate devices
